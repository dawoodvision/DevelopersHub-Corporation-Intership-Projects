# 🏦 Customer Churn Prediction

## 📌 Project Overview

This project predicts whether a bank customer is likely to leave the bank using machine learning.  
It focuses on analyzing customer data, building predictive models, and identifying key factors that influence churn.

---

## 📊 Dataset

The project uses the **Churn Modelling Dataset** from Kaggle.

### Key Features

- 💳 Credit Score  
- 🌍 Geography  
- 👤 Gender  
- 🎂 Age  
- ⏳ Tenure  
- 💰 Balance  
- 📦 Number of Products  
- 🟢 Active Membership  
- 💼 Estimated Salary  
- 🚪 Exited (Target Variable)  

---

## ⚙️ Workflow

- 📥 Data Loading  
- 🧹 Data Cleaning  
- 🔍 Checking Missing and Duplicate Values  
- ❌ Removing Unnecessary Columns  
- 🔄 Encoding Categorical Features  
- 📊 Exploratory Data Analysis (EDA)  
- 🤖 Model Training  
- 📈 Model Evaluation  
- 🔎 Feature Importance Analysis  

---

## 🤖 Models Used

- 📈 Logistic Regression  
- 🌳 Decision Tree  
- 🌲 Random Forest  

---

## 📊 Results

- 📈 Logistic Regression Accuracy: **81.05%**  
- 🌳 Decision Tree Accuracy: **78.35%**  
- 🌲 Random Forest Accuracy: **86.95%**  

🏆 **Random Forest performed best among all models**

---

## 🔍 Key Insights

- 📌 Dataset was clean with no missing or duplicate values  
- 🧹 Removed irrelevant columns like IDs and names  
- 🔄 Proper encoding improved model performance  
- 🌲 Random Forest provided the best accuracy  
- 📊 Feature importance revealed key churn factors
